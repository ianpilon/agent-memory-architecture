# Agent Memory Architecture

A five-layer memory architecture for AI agents, derived from the memory-as-navigation thesis
(Conway, Damasio, Bruner, Klein & Nichols via Youssef, Levin, and Xu et al. / NapMem for measured
engineering evidence), with a build-feasibility verdict for each layer.
